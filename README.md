# 🛒 Online Retail Data Analysis (EDA)

## 📌 Objective

The objective of this project is to analyze online retail data to understand customer behavior, sales trends, and key revenue drivers.

---

## 📊 Dataset

* Online retail transactional dataset
* Features include: InvoiceNo, StockCode, Description, Quantity, InvoiceDate, UnitPrice, CustomerID, Country

---

## ⚙️ Steps Performed

### 1. Data Cleaning

* Removed missing values
* Handled negative quantities (returns)
* Removed duplicates

### 2. Feature Engineering

* Created TotalAmount (Revenue)
* Extracted time-based features (Month, Day, Hour)
* Created customer & product-level features
* Performed RFM (Recency, Frequency, Monetary) analysis
* Customer segmentation

### 3. Exploratory Data Analysis

* Univariate analysis (distribution, counts)
* Bivariate analysis (relationships between variables)
* Multivariate analysis (time, country, revenue trends)

---

## 🔍 Key Insights

* United Kingdom dominates revenue contribution
* A small group of customers contributes most revenue
* Sales distribution is highly skewed due to bulk orders and returns
* High-value customers identified using RFM segmentation
* Opportunity exists for expansion in international markets

---

## 🛠️ Tools Used

* Python
* Pandas
* NumPy
* Matplotlib

---

## 📈 Conclusion

This project demonstrates how data analysis can be used to extract meaningful business insights from transactional data and support decision-making.
